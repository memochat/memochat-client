# Memochat client

## ✨ 시작하기

```bash
yarn install
yarn dev
```

## 🧚 스토리북

#### 실행

```bash
yarn storybook
```

#### 빌드

```bash
yarn build-storybook
```

#### 배포

- master 브랜치에 push 하는 경우 github action으로 github page에 자동 배

  - [스토리북 링크](https://memochat.github.io/memochat-client/)

- 수동 배포하는 경우 : `yarn deploy-storybook`
